# PROMPT — DESENVOLVIMENTO DA ONTOLOGIA OWL NO PROTÉGÉ

Gerar:

1) Estrutura da ontologia no Protégé
2) Classes principais:
   - eMBB
   - URLLC
   - mMTC
3) Propriedades conforme dissertação:
   - latency
   - throughput
   - reliability
   - jitter
   - packet_loss
   - coverage
   - mobility
4) Relações e hierarquias
5) Restrições e axiomas
6) Exportação para arquivo .owl
7) Integração com SEM-CSMF usando owlready2 ou rdflib
8) Validação semântica completa
