# TriSLA – Prompt Operacional

# 70 – SLO Reports

Prompt para geração de relatórios de SLO (Service Level Objectives).

# PROMPT — GERAÇÃO AUTOMATIZADA DE RELATÓRIOS DE SLO

Gerar:

1) Scripts PromQL para cálculo de SLO
2) SLO por interface (I-01 a I-07)
3) SLO por módulo (SEM-CSMF, ML-NSMF, etc.)
4) SLO do loop fechado completo
5) Geração automática em JSON
6) Geração automática em Markdown
7) Geração de PDF
8) Agendamento de relatórios
9) Comparação com targets
10) Alertas de violação de SLO
11) Dashboard de SLO no Grafana
