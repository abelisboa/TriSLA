# NASP Adapter

The NASP Adapter is the integration layer between TriSLA and the NASP environment.

## Quick Links
- [Complete Guide](NASP_ADAPTER_COMPLETE_GUIDE.md)
- [Architecture Overview](../../ARCHITECTURE.md)
- [Deployment Guide](../../DEPLOYMENT.md)

## Purpose
The NASP Adapter exposes a stable API for retrieving platform metrics and invoking platform-specific actions.

## What It Does NOT Do
- Does not make admission decisions
- Does not process semantic intent
- Does not perform ML predictions
