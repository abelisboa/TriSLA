\chapter{Mapeamento Compatível NWDAF $\leftrightarrow$ NASP}
\label{ap:nwdaf-nasp}

Este apêndice apresenta o mapeamento dos artefatos analíticos compatíveis entre a função \ac{nwdaf}, definida pelo \ac{3gpp}, e a implementação experimental \ac{nasp}.  
O objetivo é preservar equivalência semântica suficiente para alimentar os módulos \ac{ml-nsmf} e \textit{Decision Engine}.
