# 📘 Apêndice A — Catálogo de Interfaces Internas I-*

Consolida as interfaces internas (I-01 a I-07) da arquitetura TriSLA, 
incluindo protocolos, esquemas e metas de SLO, conforme descrito no Capítulo 4.

Cada interface é monitorada por métricas OTLP:

- **I-01 a I-04:** Comunicação síncrona via gRPC (Protobuf)
- **I-05:** Eventos assíncronos (Kafka)
- **I-06–I-07:** REST (OpenAPI/JSON)
