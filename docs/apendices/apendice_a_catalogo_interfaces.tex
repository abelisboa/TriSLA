\chapter{Catálogo de Interfaces Internas I-*}
\label{ap:catalogo-interfaces}

Este apêndice consolida o catálogo das interfaces internas (I-*) da arquitetura \ac{trisla}, conforme descrito no Capítulo~\ref{cap:arquitetura}.  
As interfaces I-* representam ligações \textit{intra}-\ac{smo} entre os módulos \ac{sem-csmf}, \ac{ml-nsmf}, \ac{bc-nssmf}, \textit{Decision Engine}, \ac{sla-agent-layer} e o ambiente \ac{nasp}.  
Interfaces normativas O-RAN (O1/A1/E2) não são incluídas, pois seguem as especificações da O-RAN Alliance e do \ac{3gpp}.

\medskip
\noindent\textbf{Convenções}: Classe \{sync, async\}; Protocolo \{gRPC, REST, Kafka\}; Esquema \{Protobuf, OpenAPI/JSON\}; Segurança \{mTLS, JWT/OAuth2, Políticas (OPA/ABAC)\}; Observabilidade \{OTLP traces/logs/métricas\}; SLOs (metas p99 e taxa de erro).
