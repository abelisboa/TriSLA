# 📊 Placeholder Images for TriSLA Dashboards

Este diretório contém placeholders de imagens que ilustram os dashboards esperados do TriSLA.

## 📋 Arquivos de Placeholder

Os arquivos `.txt` neste diretório descrevem o conteúdo esperado de cada dashboard. Eles devem ser substituídos por screenshots reais dos dashboards Grafana quando disponíveis.

### Imagens Esperadas:

1. **expected_dashboard_overview.png** — Visão geral dos módulos TriSLA
2. **expected_api_latency.png** — Latência da API (p95/p99)
3. **expected_pod_usage.png** — Uso de recursos dos pods
4. **expected_sla_success.png** — Taxa de sucesso de SLAs
5. **expected_resource_domains.png** — Visualização de domínios de recursos
6. **expected_blockchain_audit.png** — Log de auditoria do blockchain

## 🎨 Como Gerar as Imagens Reais

Para gerar as imagens reais dos dashboards:

1. Acesse o Grafana após a instalação do TriSLA
2. Navegue até cada dashboard
3. Capture screenshots com resolução 1093x585 pixels
4. Substitua os arquivos `.txt` pelos arquivos `.png` correspondentes

## 📝 Nota

Os arquivos `.txt` são placeholders temporários. Quando as imagens reais estiverem disponíveis, elas devem ser substituídas pelos arquivos PNG correspondentes.
