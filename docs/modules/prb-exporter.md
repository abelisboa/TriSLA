# PRB Exporter

## Overview

The PRB exporter publishes radio resource block (PRB) utilization telemetry for
RAN-aware observability.

## Role

It provides resource-pressure signals that can be consumed by monitoring layers
and, indirectly, by decision logic through telemetry pipelines.

## Data Collected

Data includes PRB-related utilization indicators and exporter health metrics
exposed for collection by observability systems.
