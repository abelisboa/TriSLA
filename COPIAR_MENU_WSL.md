# 📋 Como Copiar o Menu para WSL

## 🚀 Comando Rápido

Execute no **WSL**:

```bash
cd ~/trisla-dashboard-local

# Copiar do Windows
cp /mnt/c/Users/USER/Documents/trisla-deploy/trisla-dashboard-local/scripts-wsl/dashboard-menu.sh scripts-wsl/

# Dar permissão
chmod +x scripts-wsl/dashboard-menu.sh

# Executar
./scripts-wsl/dashboard-menu.sh
```

---

## ✅ Ou Criar Diretamente no WSL

Se preferir criar do zero no WSL, o arquivo já está criado em:
- `scripts-wsl/dashboard-menu.sh`

Basta copiar do Windows ou usar o mesmo caminho.

---

## 🎯 Primeira Execução

```bash
cd ~/trisla-dashboard-local
chmod +x scripts-wsl/dashboard-menu.sh
./scripts-wsl/dashboard-menu.sh
```

---

**Pronto! Menu interativo disponível!** 🎉
